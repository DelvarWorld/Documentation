new GUIText(id, options, guimanager, append)

Creates a new GUIText

Parameters

Name Type Description
id string The id and name element
options json Options of element
guimanager GUIManager The gui manager
append bool is added to the <body>. => True by default (optional)

Options

  • x: position left of text (in pixel)
  • y: position top of text (in pixel)
  • size: size the text (int) => 30 by default (optional)
  • color: color the text (string) => "white" by default (optional)
  • police: police font of text (string) => "Segoe UI" by default (optional)
  • text: value text label (string) => "CastorGUI" by default
  • bold: text bold or no => "none" by default else "bold"
  • italic: text italic or no => "none" by default else "italic"
  • centerVertical: text align center vertical (string) => "false" by default (optional)
  • centerHorizontal: text align center horizontal (string) => "false" by default (optional)
  • zIndex: depth of the element (int) => 1 by default
